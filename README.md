# LotoBench

LotoBench は、KYO's LOTO が配布している CSV データを元に、  
過去の抽選結果を統計的に分析し、数字の傾向を可視化・予測する **ロト分析ツール** です。

> ⚠️ 本ツールは統計的な分析を行うものであり、当選を保証するものではありません。  
> ロトの抽選は完全にランダムです。あくまで参考としてご利用ください。

## ✨ 主な機能

- ロト6 / ロト7 対応
- CSVファイル（KYO's LOTO）の読み込み
- 出現頻度ランキング表示
- 直近20回の「ホットナンバー」分析
- 出現頻度の低い「コールドナンバー」表示
- 統計情報を元にした疑似ランダム予測
- 全数字の出現率一覧表示

---

## 📂 使い方

1. KYO's LOTO から CSV データをダウンロード
2. LotoBench を開く
3. 「CSVをアップロード」からファイルを読み込む
4. 自動で分析・グラフ・予測が表示されます

また、「サンプルデータで試す」ボタンから、  
CSVなしでも動作確認ができます。

---

## 🔒 データの取り扱いについて

LotoBench は、KYO's LOTO が配布している CSV ファイルを  
**ユーザーのブラウザ内でのみ読み込み・解析するツール**です。

- CSV ファイルはどこかのサーバーにアップロードされることはありません
- 本ツールの作者が KYO's LOTO のデータを収集・再配布することはありません
- すべての処理は、お使いのPCやスマートフォン上で完結します

したがって、本ツールは **KYO's LOTO のデータを再配布するものではありません**。
